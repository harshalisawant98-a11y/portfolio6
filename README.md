# 🏪 Retail Store Project

## 📘 Project Overview
The **Retail Store Dashboard** analyzes sales performance across regions, product categories, and customer segments.  
It visualizes total revenue, profit margin, and growth trends using interactive charts built with **HTML + Chart.js** and data cleaned in **Excel**.

---

## 🧾 Dataset Summary
**Source:** Book 44.xlsx  
**Fields:**
- Date  
- Region  
- Product Category  
- Product Name  
- Units Sold  
- Unit Price  
- Total Sales  
- Customer Segment  

**Sample Data:**

## 🛠️ Tools & Technologies
- **Excel** → Data cleaning, PivotTables  
- **HTML + CSS** → Dashboard layout and styling  
- **Chart.js** → Interactive charts (Column, Pie, Line)  

---

## 📊 Dashboard Components
- **KPI Cards:** Total Revenue (₹ 1,47,600), Top Region (North), Profit Margin (20 %)  
- **Column Chart:** Monthly Sales Performance  
- **Pie Chart:** Product Category Share  
- **Line Chart:** Revenue Growth Trend  
- **Navigation Tabs:** Overview | Monthly Sales | Category Share | Revenue Growth
